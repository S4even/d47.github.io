# S4evenPortfolio GitHub Pages

Upload this `README.md` file and the complete `docs` folder to the root of a
public GitHub repository.

Then open:

`Settings` -> `Pages` -> `Deploy from a branch`

Select:

- Branch: `main`
- Folder: `/docs`

Save and wait a few minutes for GitHub Pages to publish the site.

Do not upload only a ZIP file. GitHub Pages requires the extracted `docs`
folder and its `index.html` file.
